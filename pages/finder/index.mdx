# DRAW AI System 


## 프로젝트 소개

DRAW의 AI System은 백엔드단에서 건축 도면 pdf와 찾으려는 도면 정보의 템플릿 이미지를 입력으로 받아 그에 해당하는
도면의 정보를 추출해 JSON 형식의 파일로 프론트엔드 단에 제공합니다. 도면 정보에 해당하는 요소는 도면의 사이즈, 페이지 내 존재하는 도면번호,
페이지 내 존재하는 심볼 정보 등이 있습니다 

## 지원하는 기능 

DRAW Editor가 지원하는 기능에 대한 리스트입니다.
<br />
- [x] PDF 파일 뷰어
- [x] 문서 썸네일 표시
- [x] 문서 보기 모드 (싱글, 멀티) 변경
- [x] 문서 위에 자유 그리기
- [x] 문서 위에 텍스트 입력
- [x] 문서 위에 도형 입력
- [x] 문서 위에 입력 / 삽입한 내용 지우기
- [x] 자유 그리기 펜 색깔, 굵기 변경 기능
- [x] 문서 수정 내역 다운로드 기능
- [x] AI 처리한 문서 데이터 적용 기능
- [x] 페이지 내 생성한 링크로 페이지 이동 가능
