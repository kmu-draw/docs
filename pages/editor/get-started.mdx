# 시작하기

## 사용법

다음과 같은 형태로 `<Viewer />` 컴포넌트를 임포트하고, 여러분의 앱에 마운트하여 사용할 수 있습니다. 다른 어떤 Props도 필요하지 않습니다!

```tsx filename="AwesomeApp.tsx" showLineNumbers
import { Viewer } from "draw-editor";

// some code here...

const App = () => {
  return (
    <Wrapper>
      <Viewer width="100%" height="100%" />
    </Wrapper>
  );
};
```
