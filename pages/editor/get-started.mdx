# 시작하기

## 설치하기

이 라이브러리는 비공개로 배포되어 있습니다. 따라서 라이브러리를 설치하기 위해서는 비공개 레지스트리 접근 권한이 필요합니다.

비공개 레지스트리 접근 권한이 있다면, 아래의 명령어로 라이브러리를 설치할 수 있습니다.

> 이 라이브러리는 React >= `v16.8`에서 동작합니다.

```bash
yarn add @kmu-draw/draw-editor
```

## 라이브러리 초기화

```tsx filename="AwesomeApp.tsx" showLineNumbers
import { initLibrary } from "@kmu-draw/draw-editor";

initLibrary({
  // WebAssembly URL path
  wasmPath: "/public/wasm/",
  // Event that is fired after WebAssembly module loaded
  onWASMLoaded: () => {
    // do something
  },
});
```

## 사용법

다음과 같은 형태로 `<DRAWEditor />` 컴포넌트를 임포트하고, 여러분의 앱에 마운트하여 사용할 수 있습니다.

```tsx filename="AwesomeApp.tsx" showLineNumbers
import { DRAWEditor } from "@kmu-draw/draw-editor";

// some code here...

const App = () => {
  return (
    <Wrapper>
      <DRAWEditor width="100%" height="100%" />
    </Wrapper>
  );
};
```
