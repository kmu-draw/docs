
# QATM 

### Pytorch Non-Official Implementation of QATM:Quality-Aware Template Matching For Deep Learning

arxiv: https://arxiv.org/abs/1903.07254
original code (tensorflow+keras): https://github.com/cplusx/QATM
github : https://github.com/kamata1729/QATM_pytorch/tree/master

---

### 개요 

QATM 구조는 이미지간 딥러닝 모델을 사용하여 이미지 특징 벡터를 추출하고 이를 비교합니다. 비교한 값들은 점수화를 통해 해당 심볼의 클래스를 에측합니다.
템플릿 이미지는 도면 내 찾으려는 객체 정보를 이미지로 저장해둔 것이며 도면(베이스) 이미지 내에서 이를 찾게 됩니다.


### 선정이유

QATM 은 사용자가 찾으려는 심볼만을 도면 내에서 찾을 수 있습니다. 예를 들어 도면번호가 적혀있는 심볼만을 찾기 위해서 템플릿 이미지에 해당 정보만을 
줄 수 있습니다.
